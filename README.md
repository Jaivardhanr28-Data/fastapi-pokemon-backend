# FastAPI Pokémon Cards Backend

## What is this project?
A backend API built with FastAPI that allows users to securely manage Pokémon cards using JWT-based authentication.

## Why does this project exist?
This project was created to gain hands-on experience with real-world backend development concepts such as authentication, authorization, database relationships, and safe partial updates.

## Tech Stack
- FastAPI
- SQLAlchemy ORM
- SQLite
- JWT Authentication (python-jose)
- Password hashing with bcrypt
- Postman for API testing
